# Contributing

## Building

### Windows
- Visual Studio 2019
- MSVC Toolset v142
- Configure SDK version in `src/photon.vcxproj`
- Configure paths in `copy.bat`

### Linux
- g++ 10
- g++-8-multilib
- Make 4.1
- Configure paths in `config.mk`

## Pull requests
- Follow the coding style
- Do not stage files that you had to configure

## Code style
Use the included `.clang-format` file
